GrainGuard API (1.0.0)

Download OpenAPI specification:

URL: https://github.com/pahuldeepp/GrainGuard- License: MIT

Multi-tenant agri-operations platform API. Ingests device telemetry, builds CQRS projections, and serves real-time sensor data.

Authentication

All endpoints except /health require a valid Auth0 JWT bearer token. Token must include tenant_id claim for multi-tenant isolation.

Rate Limiting

  • 100 requests per 15 minutes per IP (sliding window, Redis-backed)
  • Rate limit headers returned on every response

Devices

Device provisioning and management

Create device

Provisions a new device for the authenticated tenant. Triggers the device provisioning saga (create → attach → audit). Idempotent — duplicate serial numbers return the existing device.

Authorizations:
BearerAuth
Request Body schema: application/json
required
serialNumber
required
string [ 1 .. 100 ] characters

Unique serial number for the device

Responses

Telemetry

Real-time sensor data endpoints

Get latest telemetry

Returns the latest telemetry reading for a device. Served from Redis cache (version-keyed) with Postgres fallback. Cache TTL: version-based (invalidated on new telemetry event).

Authorizations:
BearerAuth
path Parameters

required
string <uuid>
